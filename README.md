CEN5035 software engineering project

    Nitrous
    
    Nitrous is an expansive, multidisciplinary digital ecosystem designed to serve as the definitive centralized repository for the global racing community. By integrating high-fidelity data and technical analysis from an exhaustive range of high-velocity disciplines—including Formula 1, NASCAR, and MotoGP, alongside elite National Championship Air Racing and professional Motocross—the platform addresses the logistical fragmentation inherent in modern motorsport media. Nitrous provides a sophisticated, high-performance interface featuring real-time telemetry, synchronized global calendars, and comprehensive engineering insights into both land and aerial performance machinery. Ultimately, the project aims to establish a "digital paddock" that leverages advanced data aggregation to cater to the exacting requirements of enthusiasts across the entire spectrum of velocity-based competitive sports.
